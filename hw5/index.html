Домашнее задание №5. Объекты

Создайте объект user, содержащий поле name со значением ‘John’.

  1. Запросите у пользователя ввод числа. Запишите введенное значение в поле age объекта user;
  2. Создайте копию объекта user с именем admin. Добавьте новому объекту поле role со значением ‘admin’;
  3. * Запишите все значения полей объекта admin в отдельные переменные. Имена переменных должны совпадать с названиями полей.
